## 📋 About the Project

**Money Tracker** is a simple web app for recording and tracking a user's income and outcome (expenses). Built as a midterm (UTS) project for **Introduction to Information Technology (PTI) Lab**, using plain **HTML, CSS, and Bootstrap** (no backend/database).

## ✨ Features

| Page | Function |
|:---|:---|
| 🔐 `login.html` / `hello.html` | Initial login page for the app |
| 🏠 `main.html` | Main dashboard — displays a transaction summary (income & outcome) with color-coded entries (green for income, pink for outcome) |
| ➕ `income.html` | Form for adding an income record |
| ➖ `outcome.html` | Form for adding an outcome (expense) record |
| ℹ️ `aboutus.html` | About/information page for the app |

## 🗂️ Project Structure

```
UTS-PTI-LAB/
├── login.html       # Main login page
├── hello.html        # Login page (variant)
├── main.html          # Transaction dashboard
├── income.html        # Income input form
├── outcome.html        # Outcome (expense) input form
├── aboutus.html         # About us page
└── style.css             # Global stylesheet
```

## 🛠️ Tech Stack

- **HTML5 & CSS3** — page structure and styling
- **Bootstrap 4 / 5** (via CDN) — responsive UI components (navbar, form, grid)
- Fully **client-side**, no server/database
